## 対面・サイクルの基本フロー

```
ガブリアス先発
  ├─ 有利: ステロ → じしん/がんせきふうじ
  └─ 不利: タスキ残し → 交代しステロ後出し

ゲンガー降臨
  ├─ vs 積みエース: ちょうはつ → みがわり
  ├─ vs 受け要員: ヘドロウェーブ命中100%
  └─ vs 理不尽な相手: みちづれ強制交換

アシレーヌ
  ├─ 積み技 → アンコール → クイックターン安全着地
  └─ 先制技 → ムーンフォース / うたかたのアリア
```
