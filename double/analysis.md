## ルート別成功条件

### いかりのつぼルート成立条件

```
① ユキメノコが先に動けること (速度確認)
② こおりのいぶきが急所に当たること (確定)
③ ワルビアルが場に残っていること (まもる/耐久)
④ 相手がいかりのつぼを知らない or 対策できない
```

**失敗パターン**:
- ちょうはつを食らう → ユキメノコが補助技を使えない (メンタルハーブ対策済み)
- ワルビアルが落とされる → カラミンゴかメガケケンカニに切り替え

### せいぎのこころバトンルート成立条件

```
① アブソルがバトンタッチを成功させること
② メガバンギラス/メガエルレイドが場に出ること
③ おいかぜが生きているターンに攻撃すること
```

## ダブル固有の立ち回り

### 初手の優先度

1. **ちょうはつ** (相手の積み/TR/補助を止める)
2. **まもる** (ねこだまし対策/次ターン確認)
3. **いかりのつぼ始動** (最速ユキメノコから)
4. **おいかぜ** (素早さ補強が必要な場合)

### ターン管理

| 行動 | 持続 | 備考 |
|---|---|---|
| おいかぜ | 2ターン | 発動ターンを含む |
| まもる | 1ターン | 連続使用は失敗率増加 |
| こおりのいぶき(急所) | 1回 | ユキメノコは確定急所ではない→1/8 注意 |

**注意**: こおりのいぶきは急所率が高い(ランク+1)が確定ではない。「きょううん」特性や急所率upアイテムと組み合わせが理想。アブソル(きょううん+ピントレンズ)が補助役として有用。

## よく当たる構築パターン

| パターン | 特徴 | 対策 |

| 壁+積み(オーロンゲ系) | M-Bで強化・ひかりのねんど採用増加 | ちょうはつ最優先先発 |
